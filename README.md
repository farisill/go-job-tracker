# job-radar-linkedin

Job Radar is a browser extension that helps you browse LinkedIn job postings more easily. It checks each job card for your chosen keywords and highlights the ones that are relevant to you. Itâ€™s a simple way to filter through listings and quickly notice roles that match your interests.

The extension is currently being designed and tested for Firefox.

The project is entirely written with Claude.

-- HOW TO ADD TO FIREFOX --

1. In your Firefox browser go to:

about:debugging#/runtime/this-firefox

2. Select "Load Temporary Add-on...".

3. Navigate to the extension's folder and open "manifest.json".

The extension will be working till you close you browser.
When you open it again, you will have to load again the extension.
